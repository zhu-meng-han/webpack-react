## Quick Start

* 安装

```bash
npm install
```
or
```bash
yarn install
```

* 运行

```
npm run dev
```

## 目录结构

`src` 目录下

```bash
├── assets                                    // 静态资源
│   ├── images
│   │   └── nothing.png
│   └── scss
│       ├── common.scss
│       ├── index.scss
│       └── resets.scss
├── components                                // 公用组件
├── constants                                 // 常量
├── index.js                                  // 入口
├── pages
│   ├── demo                                  // 业务模块
│   │   ├── index.jsx
│   │   └── style
│   │       └── index.scss
│   └── index.jsx
└── utils                                      // 公用方法
```
